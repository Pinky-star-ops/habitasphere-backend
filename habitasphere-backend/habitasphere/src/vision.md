# HabitaSphere

HabitaSphere is a role-based Apartment and Society Management SaaS platform.

## Vision
To simplify apartment and housing society management through secure, scalable, and user-friendly technology.

## Core Modules
- Role-based authentication
- Complaint tracking
- Maintenance payments
- Visitor approvals
- Admin dashboard

## Tech Stack
- Backend: Spring Boot
- Database: PostgreSQL
- Frontend: React (Planned)
